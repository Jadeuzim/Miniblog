Primeiro post:

Hoje, dia 28 de setembro, segunda-feira, é o primeiro dia em que marco como
o inicial no aprendizado da linguagem HTML e CSS. Pode-se dizer que esta
postagem marca o meu "Hello World" no web development do front end, algo que
considero que deva ser essencial para o futuro da empresa na Include Engenharia.

Segundo post:

Hoje, dia 29 de setembro, terça-feira, marca-se o dia das minhas primeiras
experiências com o CSS e sua função de extrema importância na estilização
do ambiente, tendo me vista que os elementos de visualização que o usuário
percebe é justamente o que diferencia um site de outro. Nota-se também a
importância do uso das tags "div" e "span", pois ambas servem para agrupar
textos e elementos, mas uma agurpa em linha enquanto a outra agrupa em blocos.

Terceiro post:

Hoje, dia 30 de setembro, quarta-feira, foi a data das integrações dos
elementos de CSS aprendidos com os elementos de HTML testados, de forma
que o website pudesse ter mais minha cara e meu perfil. Vale notar como
existem diferentes maneiras de se chegar a um mesmo resultado para se colorir
um parágrafo por exemplo, podemos utilizar uma "classe" ou uma "id"
dependendo da necessidade.
